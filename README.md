# Predictive Maintenance using Machine Learning

This project predicts the Remaining Useful Life (RUL) of turbofan engines using NASA CMAPSS dataset.

## Project Phases
1. Data Understanding and Setup  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering & Preprocessing (in progress)  
4. Model Building (up next)  

## Tech Stack
- Python (Pandas, NumPy, Scikit-learn, Matplotlib)
- Jupyter Notebook
- Streamlit (planned for dashboard)
